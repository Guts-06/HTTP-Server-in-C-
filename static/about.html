About This Project

This HTTP server was built entirely in C++ using raw POSIX system calls — no external libraries.

Tech Stack

  • C++17
  • POSIX Sockets (sys/socket.h)
  • pthreads for multi-threading
  • Linux system calls

Features

  • Multi-threaded connection handling
  • HTTP GET request parsing
  • Static file serving (HTML, CSS)
  • Multiple routes (/ /about /style.css)
  • HTTP status codes (200, 404)
